# CreatorGAINS

## Vision
CreatorGAINS is an all-in-one platform that connects creators, businesses, manufacturers, and investors across Africa. We empower content creators to monetize their work, connect with brands for sponsorships, track royalties, and manage their finances—all in one place.

## Core Modules

### 1. Creator Social Network
- Creator and business profiles
- Portfolio pages with verified accounts
- Follow system and content feed
- Live streaming capabilities
- Creator ratings and reputation

### 2. Sponsorship Marketplace
- Brands post campaigns and set budgets
- Creators apply and negotiate contracts
- AI-powered matching based on audience, geography, niche, and engagement
- Payment processing

### 3. Royalty Tracking System
- Track ownership of songs, videos, images, articles, and designs
- Manage revenue splits and licensing agreements
- Automated copyright tracking

### 4. Financial Hub
- African and US stocks, forex, crypto, and commodities
- Budget planning and savings strategies
- Tax estimation and financial advising
- Balance sheets and cash flow analysis

### 5. Business Intelligence
- Income and expense tracking
- Sponsorship earnings analysis
- Merchandise sales insights
- Monthly reports and growth forecasts

### 6. Manufacturer Marketplace
- Product registration and inventory management
- Creator-product matching
- QR code and tracking number system
- Reputation scoring based on quality and delivery

## Getting Started

### Prerequisites
- Node.js (v18+)
- PostgreSQL
- Git

### Installation

```bash
git clone https://github.com/Le-Obah/CreatorGAINS.git
cd CreatorGAINS
cd frontend && npm install
cd ../backend && npm install
```
